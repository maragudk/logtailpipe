# logtailpipe

Pipe STDIN to [Logtail](https://betterstack.com/logtail) over HTTP.

```shell
go install github.com/maragudk/logtailpipe/cmd/logtailpipe@latest
export LOGTAIL_TOKEN=123
echo foo | logtailpipe
```
